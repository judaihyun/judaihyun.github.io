JChart Constructor


# 생성자


JChart를 초기화 하기 위해서는 let chart = new JChart(context, config);
와 같이 new 키워드를 이용하여 생성한다.


- new 키워드를 직접적으로 명시하는 것을 권장.
new를 제거하여도 내부적으로 new로 생성하게 끔 처리되어 있음.

생성자
JChart(CanvasRenderingContext2D, Object);

return : JChart Object
CanvasRenderingContext2D : (필수) 드로잉 컨텍스트 CanvasRenderingContext2D
Object : (권장필수) 설정 값 ( config )
