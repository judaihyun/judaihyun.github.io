JChart Method


# update()


update() 메서드는 JChart(ctx, config)의 인자 config안의 설정,DATA를 기준으로 새로이 그려준다.
이를 이용하여 config.data에 새로이 값을 대입 후 update를 실행하여 차트를 사용한다.


example var dummy = [randomScalingFactor(),randomScalingFactor()]; //랜덤 값 배열 생성
config.data.datasets[0].data = dummy; // config파일에 생성된 값을 대입
myChart.update(); // config값을 기준으로 새로 그려주기.
