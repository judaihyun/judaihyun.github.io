# 기본 config 설정


기본적으로 config에 아무런 옵션이 없을 경우에도 표출이 되나, 의미있는 그래프를 그리기 위해서는 최소한
다음과 같은 설정이 필요하다.


note. let config[object] = {};

type[string] : line -> line chart를 의미.
data[object] : 표출할 데이터(point, label명) 객체
labels[array] : x축 label 명, data의 갯수와 동일해야 한다.
datasets[array[object]] : 실제 표출할 데이터, -50, 1.5, -30이 차트에 찍힘.
